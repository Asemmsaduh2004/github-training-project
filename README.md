# github-training-project

## وصف المشروع
مشروع ويب بسيط يمثل صفحة شخصية تحتوي على شريط تنقل وقسم للمعلومات الشخصية وتذييل للصفحة، تم بناؤه لتطبيق مهارات إدارة المشاريع والتعاون باستخدام Git و GitHub.

## التقنيات المستخدمة
* HTML5
* CSS3
* Git & GitHub

## خطوات التشغيل
1. قم بعمل Clone للمستودع.
2. افتح ملف index.html في أي متصفح ويب.
